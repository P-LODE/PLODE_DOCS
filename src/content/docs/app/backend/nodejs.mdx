Node.js는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임 환경입니다.

## 주요 특징

- **비동기 I/O**: 논블로킹 I/O 모델로 높은 성능을 제공합니다
- **이벤트 기반**: 이벤트 루프를 통한 효율적인 처리
- **NPM 생태계**: 방대한 패키지 생태계
- **크로스 플랫폼**: 여러 운영체제에서 실행 가능
- **확장성**: 마이크로서비스 아키텍처에 적합

## 설치하기

```bash
# nvm을 사용한 설치 (권장)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
nvm use --lts

# 직접 설치
# https://nodejs.org 에서 다운로드
```

## 기본 HTTP 서버

```javascript
const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('안녕하세요, Node.js!');
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`서버가 http://localhost:${PORT} 에서 실행 중입니다`);
});
```

## Express와 함께 사용

```javascript
const express = require('express');
const app = express();

app.use(express.json());

app.get('/', (req, res) => {
  res.json({ message: '안녕하세요!' });
});

app.listen(3000, () => {
  console.log('서버가 3000번 포트에서 실행 중입니다');
});
```

## 파일 시스템 작업

```javascript
const fs = require('fs').promises;

async function readFile() {
  try {
    const data = await fs.readFile('example.txt', 'utf8');
    console.log(data);
  } catch (error) {
    console.error('파일 읽기 오류:', error);
  }
}

async function writeFile() {
  try {
    await fs.writeFile('example.txt', '안녕하세요, Node.js!');
    console.log('파일이 생성되었습니다');
  } catch (error) {
    console.error('파일 쓰기 오류:', error);
  }
}
```
