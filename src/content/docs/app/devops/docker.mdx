Docker는 애플리케이션을 컨테이너로 패키징하고 실행할 수 있게 해주는 플랫폼입니다.

## 주요 특징

- **경량화**: VM보다 가볍고 빠름
- **이식성**: 어디서나 동일하게 실행
- **격리**: 애플리케이션 간 격리
- **빠른 배포**: 초 단위로 컨테이너 시작
- **버전 관리**: 이미지 버전 관리

## 설치하기

```bash
# macOS
brew install docker

# Ubuntu
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh

# 버전 확인
docker --version
```

## 기본 명령어

```bash
# 이미지 다운로드
docker pull nginx

# 컨테이너 실행
docker run -d -p 80:80 --name mynginx nginx

# 실행 중인 컨테이너 확인
docker ps

# 모든 컨테이너 확인
docker ps -a

# 컨테이너 중지
docker stop mynginx

# 컨테이너 시작
docker start mynginx

# 컨테이너 삭제
docker rm mynginx

# 이미지 삭제
docker rmi nginx

# 로그 확인
docker logs mynginx
```

## Dockerfile 예제

```dockerfile
# Node.js 애플리케이션
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./

RUN npm install

COPY . .

EXPOSE 3000

CMD ["npm", "start"]
```

```dockerfile
# Python 애플리케이션
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .

RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["python", "app.py"]
```

## 이미지 빌드 및 실행

```bash
# 이미지 빌드
docker build -t myapp:1.0 .

# 컨테이너 실행
docker run -d -p 3000:3000 --name myapp myapp:1.0

# 환경 변수와 함께 실행
docker run -d \
  -p 3000:3000 \
  -e NODE_ENV=production \
  -e DB_HOST=localhost \
  --name myapp \
  myapp:1.0

# 볼륨 마운트
docker run -d \
  -p 3000:3000 \
  -v $(pwd)/data:/app/data \
  --name myapp \
  myapp:1.0
```

## Docker Compose

```yaml
# docker-compose.yml
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
    environment:
      - NODE_ENV=production
      - DB_HOST=db
    depends_on:
      - db
    volumes:
      - ./src:/app/src

  db:
    image: postgres:15
    environment:
      - POSTGRES_DB=mydb
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
    volumes:
      - postgres-data:/var/lib/postgresql/data
    ports:
      - "5432:5432"

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

volumes:
  postgres-data:
```

```bash
# Docker Compose 명령어
docker-compose up -d
docker-compose down
docker-compose ps
docker-compose logs -f
docker-compose restart
```

## 멀티 스테이지 빌드

```dockerfile
# 빌드 스테이지
FROM node:18 AS builder

WORKDIR /app

COPY package*.json ./
RUN npm install

COPY . .
RUN npm run build

# 프로덕션 스테이지
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./
RUN npm install --production

COPY --from=builder /app/dist ./dist

EXPOSE 3000

CMD ["node", "dist/main.js"]
```

## 네트워크

```bash
# 네트워크 생성
docker network create mynetwork

# 네트워크에 연결하여 실행
docker run -d --network mynetwork --name app1 myapp
docker run -d --network mynetwork --name app2 myapp

# 네트워크 목록
docker network ls

# 네트워크 정보
docker network inspect mynetwork
```
