OpenAI API는 GPT, DALL-E 등 강력한 AI 모델에 접근할 수 있게 해주는 API입니다.

## 주요 기능

- **ChatGPT**: 대화형 AI
- **GPT-4**: 고급 언어 모델
- **DALL-E**: 이미지 생성
- **Whisper**: 음성 인식
- **Embeddings**: 텍스트 벡터화

## 설치하기

```bash
pip install openai
```

## 기본 채팅

```python
from openai import OpenAI

client = OpenAI(api_key="your-api-key")

response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": "당신은 유용한 AI 어시스턴트입니다."},
        {"role": "user", "content": "Python에서 리스트를 정렬하는 방법은?"}
    ]
)

print(response.choices[0].message.content)
```

## 스트리밍

```python
stream = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "긴 이야기를 들려주세요"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
```

## 함수 호출

```python
functions = [
    {
        "name": "get_weather",
        "description": "특정 위치의 날씨를 가져옵니다",
        "parameters": {
            "type": "object",
            "properties": {
                "location": {
                    "type": "string",
                    "description": "도시 이름"
                },
                "unit": {
                    "type": "string",
                    "enum": ["celsius", "fahrenheit"]
                }
            },
            "required": ["location"]
        }
    }
]

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "서울의 날씨는 어때?"}],
    functions=functions,
    function_call="auto"
)
```

## 이미지 생성 (DALL-E)

```python
response = client.images.generate(
    model="dall-e-3",
    prompt="우주를 여행하는 고양이, 디지털 아트",
    size="1024x1024",
    quality="standard",
    n=1
)

image_url = response.data[0].url
print(image_url)
```
