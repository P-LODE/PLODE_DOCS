PyTorch는 연구와 프로덕션 환경 모두에 적합한 딥러닝 프레임워크입니다.

## 주요 특징

- **동적 계산 그래프**: 유연한 모델 구성
- **직관적인 API**: Python스러운 코드
- **강력한 GPU 지원**: CUDA 통합
- **자동 미분**: Autograd 시스템
- **활발한 커뮤니티**: 연구에 많이 사용

## 신경망 구축

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 모델 정의
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 64)
        self.fc3 = nn.Linear(64, 10)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.2)
    
    def forward(self, x):
        x = x.view(-1, 784)
        x = self.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.relu(self.fc2(x))
        x = self.fc3(x)
        return x

model = Net()
```

```python
# 손실 함수와 옵티마이저
```
